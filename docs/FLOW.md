# User Flows

## Login → Dashboard
```
Login Page
  -> Authenticate
  -> XP Gain + Toast
  -> /main/dashboard
```

## Sign Up → Dashboard
```
Sign Up Page
  -> Validate fields
  -> Mock register (localApi)
  -> XP Gain + Toast
  -> /main/dashboard
```

## Transfer Flow
```
Select Beneficiary
  -> Enter Amount
  -> Confirm Transfer
  -> Beam animation (worker)
  -> Balance update + XP
```

## EcoAI Chat Flow
```
User message
  -> Mock response
  -> Typing animation
```

## EcoMall Purchase Flow
```
Browse products
  -> Add to cart
  -> Checkout
  -> Mock balance deduction
```

## Admin Panel Flow
```
Admin login
  -> Users list
  -> Manage cards
  -> Resolve tickets
  -> Product changes
```
